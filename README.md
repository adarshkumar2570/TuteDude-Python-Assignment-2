# TuteDude-Python-Assignment-2
This repository contains two Python programs for **Assignment 2 of Module 3**.

## Task 1 – Even or Odd Checker (Task1.py)
This program:
* Takes a number as input from the user
* Uses the modulus operator `%` to check if the number is even or odd
* Displays the result on the screen

File: Task1.py

## Task 2 – Sum of Numbers from 1 to 50 (Task2.py)
This program:
* Uses a `for` loop with `range(1,51)`
* Calculates the sum of integers from **1 to 50**
* Prints the final result

File: Task2.py
